# AI4EO-HoChiMinh-LandUse-ML

# Machine Learning Detection of Urban Expansion and Mangrove Change Around Ho Chi Minh City Using Sentinel-2

## About The Project

This project investigates whether Sentinel-2 satellite imagery and machine learning can be used to detect urban expansion and vegetation/mangrove change around Ho Chi Minh City, Vietnam.

## Research Question

Can machine learning applied to Sentinel-2 imagery detect urban expansion and mangrove change around Ho Chi Minh City?

## Study Area

Ho Chi Minh City and Can Gio, southern Vietnam.

## Planned Methods

- Sentinel-2 imagery
- Random Forest classification
- Change detection between years
- Feature importance / explainable AI

## Repository Structure

- `/notebooks` Main notebook
- `/figures` Outputs and maps
- `/data` Data instructions
- `/src` Python helper scripts

## Preliminary Observations

Initial Sentinel-2 imagery suggests continued urban densification around Ho Chi Minh City between the selected dates, while the Can Gio mangrove region remains a major vegetated coastal zone requiring monitoring.

## Selected Imagery

Two Sentinel-2 Level-2A images were selected for temporal comparison:

- January 2018 (baseline)
- March 2026 (recent comparison)

These dates were chosen because both occur during the dry season, reducing seasonal bias caused by monsoon rainfall, vegetation stress and cloud contamination.
